# Style Guide

## Colors
- **Blue**: `#1CA8CB` → `hsl(191, 75%, 45%)`
- **Light Blue**: `#E9F6F9` → `hsl(190, 61%, 94%)`
- **White**: `#FFFFFF` → `hsl(0, 0%, 100%)`
- **Dark Blue**: `#113D48` → `hsl(193, 64%, 18%)`
- **Text**: `#0D0D0C` → `hsl(60, 6%, 5%)`
- **Subtext**: `#6E7070` → `hsl(180, 1%, 44%)`
- **Border Line**: `#999999` → `hsl(0, 0%, 60%)`
